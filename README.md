## Getting Started
Follow these steps to get a local repository up and running:

1. Clone the repository on local machine:
`git clone https://github.com/Codegnan-HYD/sql-app-12.git`

2. Create a new branch for your work:
`git branch <your-branch-name>`

3. Switch to your branch:
`git checkout <your-branch-name>`

## Contributing
1. Create a file and save all the stored procedures in it:
`SP.sql`

2. Add the file to the staging area:
`git add SP.sql`

3. Commit your changes:
`git commit -m <your commit message>`

4. Push your branch to the remote repository:
`git push https://github.com/Codegnan-HYD/sql-app-12.git <your-branch-name>`
